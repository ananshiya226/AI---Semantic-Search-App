# 🚀 AI Semantic Search Engine

## 📌 Problem Statement
Traditional search systems do not understand the meaning of queries. This project builds a semantic search engine using vector databases.

## 💡 Solution
This project uses ChromaDB to store data and retrieve the most relevant result based on user query. It demonstrates a simple Retrieval-Augmented Generation (RAG) concept.

## ⚙️ Tech Stack
- Python
- Streamlit
- ChromaDB

## ▶️ How to Run
```bash
pip install streamlit chromadb
python -m streamlit run app.py
